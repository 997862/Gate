# Gate

**Gate** is a high-performance proxy management tool based on **sing-box**, designed to replace Soga.

## Features
- **Protocols**: VMess, VLESS, Trojan, Shadowsocks, Hysteria2. (Covering Soga's core protocols via Sing-box)
- **Management**: Multi-instance management via CLI.
- **Install**: One-click install script.

## Install
```bash
bash <(curl -Ls https://raw.githubusercontent.com/997862/Gate/main/install.sh)
```

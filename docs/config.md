# 配置指南

Gate 的核心逻辑是**完全由面板驱动**。

## 1. 配置文件位置

配置文件位于 `/etc/gate/gate.conf`。

## 2. 必填参数

你需要编辑该文件，填入以下信息：

```ini
# 面板 API 地址 (注意不要加末尾斜杠)
webapi_url=http://你的面板地址

# 面板通信密钥 (在面板节点设置里获取)
webapi_key=你的密钥

# 节点 ID (对应面板后台的节点编号)
node_id=1
```

## 3. 重要提示

- **不要**在服务器上手动指定端口！
- **不要**在服务器上手动添加用户！
- 所有操作请在面板后台完成，Gate 会自动同步。

# 贡献指南

欢迎为 Gate 项目贡献代码！由于本项目是纯 Bash 脚本，贡献门槛非常低。

## 如何开始

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m "Add some AmazingFeature"`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## 开发规范

- 所有的脚本请使用 `bash` 语法，确保兼容性。
- 新增功能时，请同步更新 `docs/` 目录下的文档。
- 保持代码整洁，适当添加中文注释。
